# 🌾 Crop Disease Analytics using Python & Power BI

> A Data Analytics project that explores agricultural crop data using Python for Exploratory Data Analysis (EDA) and Microsoft Power BI for building an interactive dashboard.

---

## 📌 Project Overview

Agriculture plays a vital role in food production, and understanding crop conditions through data analysis helps in making better farming decisions.

This project analyzes a crop recommendation dataset containing soil nutrients and environmental parameters such as:

- 🌡 Temperature
- 💧 Humidity
- 🌱 Nitrogen (N)
- 🧪 Phosphorus (P)
- ⚡ Potassium (K)
- 🌧 Rainfall
- ⚖ pH Value

Using Python, various statistical analyses and visualizations were performed. The processed data was then used to build an interactive Power BI dashboard for easy exploration and insights.

---

# 🚀 Project Objectives

- Analyze agricultural crop data.
- Perform Exploratory Data Analysis (EDA).
- Understand the distribution of soil nutrients.
- Study relationships between environmental factors.
- Build an interactive Power BI dashboard.
- Present meaningful insights through visualizations.
